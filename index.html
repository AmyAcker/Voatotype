General notes

Welcome to the Voat prototype, a.k.a. Voatotype.

These pages are generated from this source. Voatotype uses Jekyll for basic templating functionality.

Voatotype should work in the very latest versions of Firefox, Chrome, Safari, Opera, and Edge (including mobile versions of those browsers). To support many older versions, it needs vendor prefixes. It's best to do this with Autoprefixer, which can be used with Visual Studio and other tools. It probably also needs additional work to support IE, due to the use of flexbox. It has not been tested whatsoever on IE.

Most links on demo pages don't go anywhere. The header buttons (other than the launcher) don't yet do anything.

The launcher

The launcher allows you to quickly navigate through your subscribed subverses — or, as shown here — the default subs for logged-out users.

To open the launcher, you can click / touch the header Voat logo or type "v" on your keyboard.

To close the launcher, click on the translucent page backdrop or hit "esc".

To open the first link in the filtered list, hit "enter", or press the submit button on your on-screen mobile keyboard. "Shift + enter" goes directly to */new. "Ctrl + enter" opens a new tab.

Current limitations: clicking on the Voat logo when the launcher is open will take the user home. This currently works, but indication of this feature should be more obvious. "Shift + enter" does not currently work with random / random NSFW, but it should. There should also be information in the launcher about how to use it, perhaps shown when a "?" icon is clicked.
